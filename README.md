# **E-Crime Management System**

**A web-based crime reporting and management system** that allows citizens to file complaints online and enables police officials and administrators to manage, investigate, and track crime records efficiently.

## **<u>Overview</u>**

The **E-Crime Management System** provides a centralized digital platform to replace traditional paper-based crime reporting. It supports **role-based access** for citizens, police officials, and administrators to ensure **secure and structured crime management**.

## **<u>Features</u>**

- **User registration and login**
- **Online crime complaint submission**
- **Complaint status tracking**
- **Police dashboard for case handling**
- **Admin dashboard for system management**
- **Crime statistics visualization**
- **Role-based access control**

## **<u>Tech Stack</u>**

- **PHP**
- **MySQL**
- **HTML**
- **CSS**
- **JavaScript**

## **<u>Installation and Setup</u>**

1. **Clone the repository**


git clone [https://github.com/ssrujannn/e-crime-management-system-main.git](https://github.com/ssrujannn/e-crime-management-system-main.git)

2. **Move the project folder to your local server directory**
- **XAMPP:** `htdocs`
- **WAMP:** `www`

3. **Start Apache and MySQL** from XAMPP/WAMP

4. **Import the database**
- Open **phpMyAdmin**
- Create a **database**
- Import the SQL file from the **database** folder

5. **Open the application in a browser**

[http://localhost/e-crime-management-system-main](http://localhost/e-crime-management-system-main)
